## 2. Why-Reason-Cause

## 3. Details

# Chart Types
- Bar Chart - Comparison, trend (yang bukan berkaitan dengan masa)
- Pie/Donut Chart - Proportion, 2-5 attributes
- Line chart - comparison, trend, timeline (yang berkaitan dengan masa)
- Cards - totals, highest, lowest

# UI/UX Basic Principles
## Color
- maximum 3 colors, excluding b&w
## Font
- maximum 2 font types
## Layout
- laft to right
- top to bottom
- central (to highlight most important data in the middle)
- try to use design to guide user on how to read the dashboard
## Size, Gap, Position
- standardised accross all pages
